Le CSS, c'est quoi?

CSS: Cascading Style Sheats

  • Le concept de feuilles de style est apparu en 1996 avec la publication par le W3C d'une nouvelle recommandation intitulée « Cascading StyleSheets » (feuilles de style en cascade), notée CSS.
  • Le principe:
    Regrouper dans un même document des caractéristiques de mise en forme associés à des groupes d’éléments
  • La nécessité:
    Compenser les manques du langage HTML en ce qui concerne la mise en page et la présentation.

Le CSS, c'est quoi?

  • Les apports:
    • Une présentation homogène sur tout un site en faisant appel sur toutes les pages à une même définition de style ;
    • Le changement de l'aspect d'un site complet entier par la seule modification de quelques lignes ;
    • Une plus grande lisibilité du HTML, car les styles sont définis à part ;
    • Des chargements de page plus rapides, pour les mêmes raisons que précédemment ;
    • Un positionnement plus rigoureux des éléments.

Le CSS : utilisation

=> Avec la balise style

  • Tout dans la même page: 
    			<style type="text/css">
			<!--
			Mon code CSS
			-->
			</style>
  • Avec la propriété import:
    			<style type="text/css">  
			<!--  
			@import url("source/page.css");  
			--> 
			</style>

Le CSS : utilisation

=> Avec une balise meta (avant la fin de head)

  • 			
    
			<link rel="stylesheet" type="text/css" href="css/style.css" media="all"/>

Les animations 
			div
			{
			width:100px;
			height:100px;
			background:red;
			position:relative;
			animation:myfirst 5s;
			}

			@keyframes myfirst
			{
			0%   {background:red; left:0px; top:0px;}

			50%  {background:blue; left:200px; top:200px;}
